Avant de compiler et de tester ImacStadium, veillez à ce que la librairie javax.json (situé dans le dossier 3Party) soit bien ajouté à vos librairies de compilation.
Pour cela, ouvrez le projet ImacStdium dans Eclipse, puis allez dans :

Project -> Properties -> Java Build Path -> onglet Libraries

Si vous ne voyez pas javax.json-1.0.2.jar, ajoutez-le en cliquant sur Add JARs et allez chercher le .jar situé dans 3Party, puis Ok.
